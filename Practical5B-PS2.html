Sum Of 10 Numbers

Code For Given Problem Statement

Problem Statement

Write a program in any language to read 10 numbers from keyboard and find their sum and average.
				
					using namespace std;
					int main()
					{
						int no,sum=0;
						float avg;
						cout<< "Enter Numbers:";
						for(int i=1;i<=10;i++)
						{
							cout<< "Number is:" << i;
							cin>>no;
							sum+=no;
						}
						avg=sum/10.0;
						cout<< "Sum is"<< sum;
						cout<< "Average is:"<< avg;
						return 0;
					}
